# twicker

## Key Features

  * Sends asynchronous requests to Twitter API to get new headlines from 3 different sources
  * Scrolls headlines infinetly across the screen
  * Animation stops when mouse hovers over headline
  
## Technologies

  * JavaScript
  * Node (Express)
  * jQuery
